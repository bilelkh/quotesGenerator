# Quotes Generator

## Setup

The first thing to do is to clone the repository:

```sh
$ git clone 
$ cd quotesGenerator/frontend/
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
